I believe you refer to Go Moku, not Go. They are both played on a Go board but
only Go Moku is an x-in-a-row game.

X-in-a-row games where x is also the length of a dimension get more drawish as
x increases, but less drawish as the number of dimensions increase. These games
are also much more drawish when x is even. For games that are 'balanced' (in
that it is trivial nether for the first person to win nor for the second person
to draw) will generally have the second player playing defensively for the
whole game.

OTOH, games like Go Moku, where the length of a dimension (19) greatly exceed x
(5) allow for more ebb and flow, given the limits of human calculation.
