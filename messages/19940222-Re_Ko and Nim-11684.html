The super ko games don't sound very interesting.  Because of the
extremely large number of possible moves, the games will be very long
and only end when all possible moves are repeats.  I don't see any
method of analysis which is likely to work short of searching an entire
game tree.  (And because the state of the game includes the list of
positions which have occurred before, that game tree will be very large
indeed.)
