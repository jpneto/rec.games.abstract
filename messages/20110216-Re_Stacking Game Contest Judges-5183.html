How about the actual judging process?  Someone already mentioned this
system, or something like it I think, but here's the idea: Each of 3 judges
has 100 points to work with, and they can distribute those points to the
games in any manner they wish, so long as all the game scores total 100 for
each judge.  Then add up the scores for each game.  If there's a tie, each
judge votes for the single game of his choice.
