One of my friends came up with this nice position, with only 2 moves left:

   a b c d e f g h
1 x x x x x x x x
2 x o o o x x x x
3 x o o o o o x o
4 x o o o o x o o
5 x o x o x x o o
6 x o e o o o x o
7 x x o o x x o o
8 ? e o o o o o o

x = black
o = white
e = empty

It is whites turn. If the corner a8 is white, then white will lose If the
corner is black, then black will lose.
