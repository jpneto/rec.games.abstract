More on Neutreeko: I've decided to introduce a new starting
position, which is easier to remember and convenient in OTB
matches:

    +---+---+---+---+---+
  5 |   | o |   | o |   |
    +---+---+---+---+---+
  4 |   |   | x |   |   |
    +---+---+---+---+---+
  3 |   |   |   |   |   |
    +---+---+---+---+---+
  2 |   |   | o |   |   |
    +---+---+---+---+---+
  1 |   | x |   | x |   |
    +---+---+---+---+---+
      A   B   C   D   E

I hope not too many get upset for not being able to use
their openings, esp. with a tournament coming up in just
over three weeks. Actually that shouldn't be a huge problem
since perfect play from the old and the new starting
position leads to the same position after a few moves, only
with colours reversed.
