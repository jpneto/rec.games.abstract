No, this would be absolutely hopeless.
Almost every game would produce something like this stand-off:

- X X X X X -
- X - O - X -
O O - X - O O
- - - X - - -
- - - X - - -

...alt-ascii...

-  X X X X X -
-  X - O - X -
O O - X - O O
-  -  - X - - -
-  -  - X - - -

X is connected to N & S, O to E & W.

Now neither dare play in the four centre blank cells,
or else the other will grab the 4th cross-point and
get connection his own way.  SO the game would
painfully run down till someone was forced to crack,
and reduce the whole thing to a Nim-like triviality.

Any connection game where there are "cold" moves
like this will do the same.
