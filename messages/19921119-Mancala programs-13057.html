This is a list of programs available via anonymous ftp which (sound like)
they play some form of Mancala.  Not having access to a Mac, Atari, or IBM,
I haven't been able to check this.  This is basically the output from a few
archie searches, tidied up, and only one site given for each country.

If you pick up one of these, could you post comments?
Do you know of any other mancala programs; any for UNIX?

One interesting item:  prolog source for a kalah player can be found at
wilma.cs.brown.edu [128.148.31.66] in /pub/pvh/benchmarks/kalah.p
I'm not conversant in prolog, so can't comment on it.

Neil
