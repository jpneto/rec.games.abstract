There is a six-cell selection which fits on the
5x5 board but actually "covers" the 6x6 board:

+---+---+---+---+---+---+
| X |   | X |   |   |   |
+---+---+---+---+---+---+
|   |   |   |   |   |   |
+---+---+---+---+---+---+
| X |   |   |   | X |   |
+---+---+---+---+---+---+
|   |   |   |   |   |   |
+---+---+---+---+---+---+
|   |   | X |   | X |   |
+---+---+---+---+---+---+
|   |   |   |   |   |   |
+---+---+---+---+---+---+

Thus the minimum for size 6 can be lowered
to 6.  I wrote a Prolog program to solve
such puzzles, and it finds the solution
above in about 10 minutes.  I'm making
some tweaks to how it intializes the
board for speed, because you have to
search in some fashion the entire space
to confirm no smaller solution exists.
