I recall reading somewhere long ago -- probably in one of
Martin Gardner's columns on polyominoes -- a suggestion for
a 2-player game using pentominoes.  Take a set of the 12
pentominoes, and an 8x8 board.  The players take turns
placing any as-yet-unused pentomino on the board.  If a
player cannot fit any of the remaining pentominoes on the
board, he loses.

This game is obviously a forced win for one player or the
other.  Does anyone know if it was ever determined which?
