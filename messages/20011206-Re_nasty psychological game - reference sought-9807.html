Nasty indeed! I tried to analyze a version with only two lines,
and any real number between 0 and one's score as a possible bid.
So what is B's probability of winning in the following position
assuming perfect play?

    |  |       Score
  A |  O B     A: 1
    |  |       B: x

Let's call it p(x). Clearly p(x) = 0 for x < 1/2, and p(x) = 1
for x > 1. I found it tricky to work out p(x) for 1/2 < x < 1. 
Eventually I came up with p(x) = 2 - 1/x, but it's not really
a proof (yet). Is it right, though?
