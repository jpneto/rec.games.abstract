These are the Round 8 Fantasy Rule Valid Ordinances: 


Ordinance 8001 (Chatham):

Rule of Hypocrisy:
Any subsequently posted fantasy rule must violate any and all 
restrictions it would impose on future fantasy rules.

Ordinance 8002 (Chatham):

Each subsequent fantasy rule A must include an example of a fantasy rule
B that would be valid if A were valid and this rule (this very rule, the
one you are now reading) had not been posted.
                        
Ordinance 8003 (Davidb):

Any future Fantasy Rules Committee Ordinances must be written using words
of only one syllable.  The 8002 mandated example is:
         All new rules must have the word "smile" in them.

Ordinance 8005 (Ferro):

 From now on, no rule may state a word more than once.  To wit, "No new
rules can have the word 'smile' in them."  Note that this one says "no"
and "word" three times each.
