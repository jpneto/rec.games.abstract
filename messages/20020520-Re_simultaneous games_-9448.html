Olivier Besson wrote:
Does anyone know simultaneus, turn-based games?
("simultaneous" <=> each player makes a "move" in secret, then moves are
performed).
Besides "Diplomacy" and "Shi-Fu-Mi", I have no idea...

On my email server there are a couple.

Tug-of-War.  Each player starts the game with 1000 points.
each turn you "bid" one or more points.  if you win, you move one closer
to the goal.  If you lose, you move 1/N (where N in the number of other players)
away from the goal.

If there's a tie for the winning bid... no movement.

All bids are deducted from each players' points, not just the winner's.

First player to reach the goal wins.  If you have no points left.  You lose.

Only draw condition is if the last 2+ players run out of points simultaneously.


I also run a Risk variant ('Risque' to keep the lawyers happy) that is played
simultaneously.
