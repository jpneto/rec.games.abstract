Generic Game Framework in eiFFel v.0.7.3 is now available.

About G2F3
----------
Generic Game Framework in eiFFel (G2F3) aims to simplify the
development
of two player zero sum strategy games by separating game specific code
from reusable generic code. So far, abstract board games are best
supported.

Release Notes
-------------
Mark Steere's strategy game Tanbo has been added to the example games.
Some board games can now be built for different board sizes. Three bugs
have been fixed (two of which were really embarassing) and the code has
been cleaned up in several places. This release requires a recent
snapshot of SmartEiffel-2.2.
