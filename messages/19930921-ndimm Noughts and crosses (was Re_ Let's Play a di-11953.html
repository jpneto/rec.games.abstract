and is it the same set as all values of (N,D) for which the
first player of an N^D noughts and crosses game cannot force a win?

Is there a function F such that a draw can be found for
{(n,d) : 0<n<F(d) } ?

I think a sufficient but not necessary (eg 3^2) condition for
a draw to be possible is that there be at least twice as
many 'squares' (points in the game space) as winning runs,
ie

n^d >=2*(  ((n+2)^d-n^d)/2)  )
