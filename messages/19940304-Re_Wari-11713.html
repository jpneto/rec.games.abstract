  There are probably several computer versions of this game, but one
is called Kalakh, available for PC's running MS-DOS.  It uses coins
instead of beads, but it is a Mancala game.  It should be available
on any MS-DOS ftp site, or I could upload it to the rec.games.abstract
FTP site (if it still exists).

  If anyone has an X11 version of Mancala, I would appreciate it.

  For another variation, see the book New Rules for Classic Games.  There
is a game called the Glass Bead Game which is interesting.  Actually,
maybe I'm wrong and it's in the Gamut of Games.  Anyway, Mancala games
are very fun to play in my opinion.
