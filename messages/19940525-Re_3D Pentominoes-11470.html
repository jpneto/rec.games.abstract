Here are the number of 3-d polyominoes:

# squares		Number (3-d rotates)	Number (4-d rotates)
---------		--------------------	--------------------
1			1			1
2			1			1
3			2			2
4			8			7
5			29			23
6			166			112
7			1023			607
8			6922			3811
9			48311			25413

3-d rotates is probably what you want if you're talking about the
physical pieces, since we live in a 3-d world (or so it seems =) ).
The 4-d rotates number means if one piece can be flipped through the
4th dimension to get another, you only count them once.  The only reason
this definition makes any sense is that when we talk about 2-d polyominoes
we usually talk about the 3-d rotate version.

If you want larger numbers I can probably try to compute them...
