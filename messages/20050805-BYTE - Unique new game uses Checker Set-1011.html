Subject: BYTE - Unique new game uses Checker Set
From: a@bc.com
Date: 05/08/2005, 20:43
Newsgroups: rec.games.abstract

Byte is a unique new game which uses a standard Checkers set.  Byte is a
robust alternative to Checkers.  No draws or ties can occur in Byte. 
There's no jumping, capturing, killing or any of the usual ho-hum
mechanisms.  Players cooperate to form three stacks of eight.  Winner of the
majority of the three stacks wins the game.  Byte was invented by Mark
Steere on July 13, 2005.  See bytegame.com for the rule sheet.
