The generalized rule was suggested by Luis and implemented in Scware:
http://mindsports.nl/index.php/arena/scware/

There it reads: "At the end of a player's turn any two diagonally adjacent like-colored stones must be part of the same group", where a 'group' is already defined as orthogonally connected.

Luis gives a number of reasons to use the non-generalised rule in this thread:
http://www.boardgamegeek.com/thread/1081423/new-connection-games-with-the-slither-restriction
