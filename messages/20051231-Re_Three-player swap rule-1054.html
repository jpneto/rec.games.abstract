On 2005-12-31, camb <cameronb@optusnet.com.au> wrote:


I'm wondering whether this might suggest an optimal swap rule for
three-player games with a first (and possibly second) move
advantage? Is there currently a generally accepted swap rule for
multiplayer games?


You probably can't do it in general, because the game mechanics may not
support handicapping in a way that will work. It also isn't generally
as important as if one player gets a better initial position, the other
players can generally collude to neutralize that advantage as the game
is being played.
