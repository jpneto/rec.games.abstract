Actually, on a full NxN board, it is a win for the second player
unless N=1 (in which case, it is obviously a first-player win).

If the board is initially full, it doesn't matter which row or column
you take, as they are all alike.  This is true even after a number of
rows or column are taken.  Hence, all that matters is whether you take
a row or a column.  So we may as well assume you always take the
leftmost column or topmost row.  If you start with a square, the
second player can (by taking row when the first player takes column
and vice-versa) always force it into a square with one less colunm and
row, i.e., from NxN to (N-1)x(N-1) etc.  So, if N>1, the first player
is eventually faced with a 2x2 board.  If he removes a row or column,
the second player removes the other and thereby wins.
