The anonymous FTP archive for rec.games.abstract is now available.  This
is intended to be a repository for useful information and code culled
from r.g.a and elsewhere which may be of interest to readers of r.g.a.
The initial toplevel README is included below.

If you don't have anonymous FTP at your site, you can access the archive
using an FTP mailserver.  Send email to

	ftpmail@decwrl.dec.com

with a message body containing the single word: help
The subject line of your message is ignored.

		Anonymous FTP Archive for rec.games.abstract


The ftp archive for rec.games.abstract is at agora.scs.leeds.ac.uk
(currently 129.11.144.129, but this will change; the name won't).

To connect to the archive (on an internet connected unix box) type:

	% ftp agora.scs.leeds.ac.uk

Login as `anonymous', giving your full email address (eg neilb@scs.leeds.ac.uk)
as the password.  Once in, change directory to /pub/rec.games.abstract, eg:

	ftp> cd pub/rec.games.abstract

The archive is organised as follows:

	README		This file

	books		books of interest, and other bibliographic material

	games/		directory containing rules, strategies etc for games,
			either with a single file per game, or a directory
			containing a number of files related to the game.

	games/INDEX	list of games.

	programs/	any programs not available via ftp elsewhere.

Any comments and suggestions related to the rec.games.abstract archive
should be emailed to the address given below.
