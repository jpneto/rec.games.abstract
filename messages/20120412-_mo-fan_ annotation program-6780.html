I've created an Axiom program that I believe can be used to annotate
many games (such as Oust) that can be played on a 19x19 go board.

You can download it from here if you want to try it out:
http://www.datafilehost.com/download-2eb54075.html

After installing, launch the program "mo-fan" and you can edit the
board by right clicking on an intersection and selecting from a list
of pieces and indicators.

(the name "mo-fan" is derived from the phonetic Chinese equivalent of
the word "pattern").
