Richard's Play-By-eMail Server (pbmserv@netcom.com) is pleased to announce
the addition of the game, Hex, to the server.
 
        | challenge <Vert-userid> <Horz-userid> [ <handicap> ]   |
        | standings                                              |
        | show <board#>                                          |
    Hex | move <board#> <userid> <password> <move>               |
        | resign <board#> <userid> <password>                    |
        | preview <board#> <move>                                |
 
    On a 11x11 board hexagonal, players take turns placing stones of
    their color on the board. The object is to connect your sides of
    the board while preventing your opponent from doing the same.

Send pbmserv a message with 'help' (minus the quotes) as the subject
for more information.
