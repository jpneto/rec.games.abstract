On www.littlegolem.net will start First Little Golem Championship (go
19x19) on 22nd April. Tournament is opened for all players.

Little Golem (www.littlegolem.net) is server for playing board games.
Features:
 - web based (no download, no java, no problems with firewall)
 - turn based (approx. 1-2 moves/day)
 - several kinds of games (go,chess,hex,amazons,reversi)
 - rating
 - email notifications
 - export games in SGF,PGN
