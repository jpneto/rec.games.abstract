Not sure if anyone on this forum already knows this, but I found a set
of rules for some simple abstract games using up to several sets of
checkers and a single checker board:

Go to talk.consimworld.com. In the drop-list at the top, choose the
category "Euro, Family Games, Other". Scroll down the resulting list
until you see the link to Ty Bomba's New Millennium Checkers. Once in
the folder, you'll see a link to the .rtf document with the rules.

The game is actually 4 sets of rules to 4 different games. Most require
multiple sets of checkers, but all are played on a standard 8x8
checkerboard.
