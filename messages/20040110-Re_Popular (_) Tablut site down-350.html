Quoting Tom Granvold's message of Yesterday:


So you know of any other site where Tablut can be played?


Since expomedia went down, I look at http://www.ludoteka.com/ which has
Tablut among a number of other games.  I think it was mentioned either
here or in rec.games.board.  Ludoteka is primarily for games between
human opponents though, and the computer player is very weak.  It helps
you get a feel for the game if you've never played before, but I'm still
learning and it provides me no challenge.

I'm now getting my daily tafl fix from the old DOS game Viking Siege,
which is the best tafl game I've found to date.  Tablut isn't one of the
tafl variants it plays, but it's freeware so I don't complain.  There's
also a Windows game Tabula Letum which I think *is* tablut, but I'm
running these things on Linux using emulation, and Tabula Letum doesn't
like the idea at all.
