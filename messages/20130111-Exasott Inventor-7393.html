"The game could be played with pencil and pre-printed paper.

Print a board 8x8 with circles instead of tokens.
Print 16 sticks in front of each player with a circle at right

That is it!

For each game you use one sheet.
You could even use only one sheet if instead of cross you just place a stone.

Hmmmm.  Hazen's game of Exasott was purportedly implemented by someone he doesn't know.  Odd because Combinatorics123 does know the programmer of Exasott - Alex Hammel.  He met Alex in the xkcd forum while "BEGGING PROGRAMMERS TO IMPLEMENT [his] GAME".

There's no executable program that I could find for Exasott (https://github..com/ahammel/Exasott).  Just a bunch of raw code.  Exasott can't be found on any game servers.  So you can only test Exasott by yourself apparently if you have a Python interpreter.  

A crappy program for a crappy game by a crappy inventor.
