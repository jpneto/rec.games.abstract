pbmserv@gamerz.net is an automated play-by-email server that (at last count)
understand over 130 various (most abstract) games.  We have 2000+ users scattered around the world, and the server processed ~3000 moves a day.

It does NOT play the games.  Rather, it acts as referee.  Players send their moves to the server (as the Subject: line), and it verifies that the move is valid for the game/board, and it updates the state of the board and sends the new board to all interested parties.  (people can subscribe to games in progress, etc.)  Additionally, any text included in the BODY of the message is included with the board as comments.  So you can have nice, long running conversations with your opponents.
