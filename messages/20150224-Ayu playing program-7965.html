FoulOleAyu is an Ayu playing program which placed second at the 2015 CodeCup challenge with 138 out of 150 points, just 4 away from the winner.

His author, Marc Oldenhof from the Netherlands, was so kind as to send it to me together with a functional GUI. I'm sharing it here with his permission:

https://drive.google.com/file/d/0B3gpl2WT8ufWU0N0Undyemt3Y2s/view?usp=sharing

I hope you'll like it.

Notes:

- The menu button at the bottom right might not work properly the first time, in which case simply clicking anywhere and trying again solves the problem.

- In the settings dialog, some values might cause the program to crash.

- On Windows 7, it might be necessary to install Visual C++ Redistributable for Visual Studio 2012 Update 4 (http://www.microsoft.com/en-us/download/details.aspx?id=30679).

Please feel free to comment and let me know if you encounter any other problems.
