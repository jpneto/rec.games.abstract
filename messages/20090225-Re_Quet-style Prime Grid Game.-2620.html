The Basic Game
==============
Played on a 5x5 grid, the players alternately fill any grid cell
with a number from the set {1,2,3,...,29,30}.  The numbers
are on their own discs, so each one can only be used once.
(The 5 left-over numbers are discarded unscored.)

Scoring:  Each row and column is added up, and the score for that
"""""""   row/column is the highest prime factor of the total.

A scores the rows, and B the columns.  Best total of 5 wins.
