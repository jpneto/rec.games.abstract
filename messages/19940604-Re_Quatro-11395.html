	Set was invented by a programmer named Marsha Jean Falco and was pub-
lished in 1991 by Set Enterprises. I don't know who invented Quarto!, but I do
know it was published in 1991 by Gigamic S.A.

	The Set folk have come up with some additional rules for game varia-
tions. They will soon be coming out with an elaborated version of Set which is
called Triology.
	Gigamic is releasing a game called Pyraos which doesn't really resem-
ble Quarto!. It has to do with stacking balls on a pyramid -- that's all I'll
say. Like Quarto!, it is a pure strategy game which doesn't take long to play.
