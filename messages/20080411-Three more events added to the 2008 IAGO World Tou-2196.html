These three events have been added to the 2008 IAGO World Tour:

CV Potluck 2008
This begins April 14, 2008. It is a series of Chess Variants in an
Athlon format. It is run by ChessVariants.com .

Boardspace.net Summer 2008 Hive + Mosquito Tournament
This event begins June 1, 2008. The creator of Hive will be awarding a
trophy and introducing the new Mosquito piece.

Boardspace.net Zertz Beginner's Tournament
This begins June 1, 2008. It is a beginner's tournament for the game
Zertz. It is run by Boardspace.net .

For the full schedule, and direct links, visit: http://www.IAGOWorldTour.com
