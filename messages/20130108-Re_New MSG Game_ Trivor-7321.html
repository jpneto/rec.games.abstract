Seeking perfection in games is misguided.  Perfection is unnecessary.  Ideally, a game is scalable, such as Oust, which is usually played on a 7x7x7 board.  Turn order advantage is statistically consistent with zero percent.  Someday, if players become expert enough, and if turn order advantage emerges, we may have to switch up to an 8x8x8 board, bringing turn order advantage back down to virtually zero.  

In order for a game like Oust to be perfectly balanced, the board would have to be infinitely large, making it a non-game.  You want the board to be large enough to where turn order advantage is negligible, but small enough to where the game doesn't take forever to play.
