You have 10 stacks of 10 coins.  One whole stack of coins is counterfeit.
Each real coin weighs 10 grams.  Each counterfeit coin weighs 11 grams. 
You have the use of a scale ( not a balance ) that reads weight in grams. 
Using only one weighing, how can you determine which stack is counterfeit.


Put the following coins on the scale:  0 coins from the first stack, 1 coin
from the second, 2 from the third, 3 from the fourth, ...  Assuming the
scale is very accurate one could then determine which stack is counterfeit
by the amount the weight deviated above a 10 gram multiple.  For example, if
the fourth stack was counterfeit the total weight would be 103 grams.
