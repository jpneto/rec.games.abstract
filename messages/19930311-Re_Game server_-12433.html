There is a bridge server, of sorts.  Actually what happens is that there
is a global server which is just a database of running table servers.
Each player contacts the Global Player Service (GPS) to see if there is
a table to join.  If not, the player may spectate at another table or
become a new table server.  "Serving" registers the new table with the
GPS so that new players may join.  More information may be obtained by
rooting around the following FTP sites or asking on rec.games.bridge.

--------------------

Frequently Asked Questions about OkBridge
---------- ----- --------- ----- --------

1. What is OKbridge?

   OKbridge is a Unix program which allows four people at different
   locations on the Internet to play the game of bridge together.
   It has Rubber, Duplicate, Match Point and IMP scoring modes.
   It allows kibitzers to join a table and watch play, it provides
   a practice mode whereby a partnership can practice bidding,
   and it provides support for duplicate play.
