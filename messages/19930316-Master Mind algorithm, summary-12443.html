Last month I posted a request for an efficient algorithm for the 
code-breaker player in Master Mind.  Here is a summary of the replies.

Lucas B. Kruijswijk sent me C code implementing an algorithm to solve 
6-color, 4-position game in 5 moves.  I would give his email address, but 
he is moving soon and his address will change.

Thomas Hoppe writes that an algorithm implemented in Prolog has (probably) 
been published in one or more of the following:
  M. van Emden 1978, Relational Programming, Research Report CS-78-48,
    DCS, Univ. of Waterloo, Canada
  R. Kowalski 1979, Logic for Problem-Solving, North-Holland
  H. Coelho, J.C. Cotta, L.M. Pereira, 1980, How to solve it with Prolog,
    Tech. Rep., Laboratorio Nacional de Engenharia Civil, 2nd ed., Lisboa
  H. Coelho, J.C. Cotta, 19??, How to solve it with Prolog, Springer-Verlag.

Brian Schott writes that Roger Hui recently published an algorithm 
implemented in the language J in the newsgroup comp.lang.apl.

  Thanks to all respondents.
