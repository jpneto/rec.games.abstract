Consider the following Nim games:



A) A number of pieces (all of the same color) are placed on a NxM
board. No field contains more than one piece (one might relax this
condition, but I don't believe it adds more 'depth' to the game). Two
players in turn make a move which consists of taking one or more
pieces from the board. Pieces taken in each move should come from the
same horizontal or vertical row. The player who made the last move
wins.


If N and M is odd then it is a win for the first player else it is a
win for the second player.
