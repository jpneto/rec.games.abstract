On Tuesday, March 10, 2015 at 4:45:15 PM UTC-4, Moh Bel wrote:

Easy to create a chess variant.
You play chess with all the classical rules but you add a new capture mechanics.
Queen can not capture an opponent queen
Rook can not capture an opponent rook
Bishop can not capture an opponent bishop
Knight can not capture an opponent knight 
Pawn can not capture an opponent pawn
King can be adjacent (orthogonal and diagonal) to the opponent king.

All the gameplay will change! 
Why? Because pieces are going to be used as blocking pieces.

Try it! 

I hope you will like it.
