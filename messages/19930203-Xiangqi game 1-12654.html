The following game was played in the first Finnish Open in Xiangqi
in January 1993. The game is meant for you to analyse! Please post your
analysis to this group.

I have used short algebraic notation which is the best in my opinion
(but not necessarily are the names of the pieces: Rook,Horse,Elephant,
Assistant,General,Cannon,Soldier). Rank numbers are 1-10, file letters a-i.
I have heard requests to number the ranks 0..9 (or 1..,9,0) - for easier 
programming or for compatibility with existing xiangqi programs? As for me, 
I don't think this should have any influence in the notation system.
The initial position is

10  RHEAGAEHR
 9  .........
 8  .C.....C.
 7  S.S.S.S.S
 6  .........
 5  .........
 4  S.S.S.S.S

    abcdefghi
