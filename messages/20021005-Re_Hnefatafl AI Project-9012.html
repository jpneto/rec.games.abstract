That sounds like it'd be tough. I'm glad I'm not a computer programmer!

One piece of advice: the attacking side must watch out for the possibility
of the defenders opening up 2 lanes for the king to escape at once. If this
happens, it's game over since the attackers can only move one man and block
one route at once. Not sure how that translates into "zeros and ones", but
maybe it'll help?

There is a game called HiQue that uses some similar mechanisms to tafl
(well, really only one: custodial capture). At the following link, you can
read about a similar exercise for a computer class involving that game. It
might give some insights, even though it's not the same game (it's a good
game in its own right, IMO):

http://www.math.uaa.alaska.edu/~afkjm/cs405/hique/hique.html
