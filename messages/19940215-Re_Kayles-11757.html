dbush@csugrad.cs.vt.edu (David Bush) wrote:


Does someone have the rules for the game called Kayles?



I believe I know a game by that name.  You have n tokens in a line, and each
player alternately removes either a single token or a pair of touching tokens;
whoever takes the last token wins.  Is that the one you're thinking of?


It sounds like the first player could always win by dividing the line into
2 identical regions on the first move. If n is odd, remove the middle token.
If n is even, remove the 2 middle tokens. Now any move the 2nd player makes
is answered by the corresponding move in the other region. The 1st player
must always win with this procedure, n'est-ce pas?



That's correct. Positions with consecutive pins are all first-player wins.

But the game gets interesting with more varied initial positions.
For instance, consider the position with two rows of consecutive pins,
of length 4 and 995. What is a good move now?

Those familiar with Winning Ways should be able to solve this one
(having a computer handy doesn't hurt:-).
