Played against Mark today. Similar to No-Pass Go (and hence to Go),
but the red stones create many interesting situations. For instance,
you can't save a group by joining it to a living group if filling the
only shared liberty results in a capture which doesn't ensure life by
itself. Also, while simultaneous capture wouldn't be advisable in Go
because of cycles, here it works nicely and increases the tension.

IMO, both features together help make Red Stone a bit more balanced
than Go, even if the pie rule is still essential.
