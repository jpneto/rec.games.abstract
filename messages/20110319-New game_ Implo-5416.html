Some time ago, I asked myself: "What if Go was played such that
capturing groups were captured?" I felt that question hid an
interesting game yet to be discovered. Implo is the result of that
idea. It's a simple, finite, drawless game without the need for ko or
superko rules, and it's got a high stone-recycling ratio. It's fun to
discover how strangely different it feels from Go while still ringing
a familiar note.

It features a (new?) capturing mechanism for naturally finite games.
In a general way, it consists of some stones being captured while, at
the same time, some others become explicitly invulnerable.

(Note: In December, in Steere's Rive thread, I mentioned a first draft
of the game which didn't quite please me. The game has quite changed
since, and it's much better now.)
