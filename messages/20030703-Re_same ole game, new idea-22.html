*The binary code used to make the characters in the core game design
is simple, and the mathematics behind the reproduction of the new
character is simple. I thought about the breeders in my two player
game and the role of sex in the multi-player games, and, knew that the
code for such a game would be staggering. The code for such characters
would be very much like DNA but represented in binary code, e.g.,
the code is for two characters, one male, and one female. The code is
for only two traits, and it is best to have model of DNA in mind to
understand my idea.
