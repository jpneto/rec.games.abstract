Subject: Why the "swap" rule is not enough to fix Bridg-It
From: David J Bush <twixt@cstone.net>
Date: 02/08/2005, 17:04
Newsgroups: rec.games.abstract

The BoardGameGeek page for Bridg-It now has a file for download
which shows why the "swap" rule (also known as the "pie" rule)
is not enough to turn Bridg-It into an interesting game.
For each possible initial move (except a move which connects
two pegs in the same border row, which might not even be legal),
all the moves on the board have been paired off, so the first
player always has a simple winning strategy. Included is also
an informal proof that these pairing diagrams work.
