What is the object of Trax?  The game I have seen that uses tiles like
these is a pencil&paper game described in Winning Ways v.2 (but I forget
what they call it).  The play of that game is: Draw a rectangular grid
on a piece of paper and mark one side of one "edge" square of the grid
as the "entrance".  Each player takes turns drawing one of the tiles
into a square of the grid so as to extend the path entering the grid
from the entrance.  The first person to cause the path to exit the grid
loses.  There is a winning strategy for the first player on grids
containing an even number of squares, which is described in the book.
