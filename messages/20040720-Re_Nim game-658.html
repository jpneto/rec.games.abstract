torbenm@diku.dk (Torben Ægidius Mogensen) wrote in message news:<7zk6wyzrgj.fsf@pc-032.diku.dk>...


How do you find these Groundy numbers?  Is it just the number of X's?


No, it's not simply the number of X's. For instance, the Grundy value
for

(X X)
(X X)

is 0. For non-trivial impartial games, computing Grundy values
constitutes a tedious task (per definition: if this computation would
be easy, the game would be trivial!), and the game under discussion is
no exception.

The Grundy number for any position is a non-negative integer defined
recursively as:

1) the end-position (empty board) has a Grundy value of 0.

2) a position with a non-zero Grundy value has some follower with any
smaller Grundy value
 

Can you prove [that Grundy values of non-interacting block positions 
can be added together using Nim-addition], or is it just a hypothesis?


Non-interacting blocks constitute 'parallel games' the same way as
individual Nim-heaps in the standard game of Nim constitute 'parallel
games'. The only difference between standard Nim and the game A)
described above, is that an individual Nim-heap constitutes a trivial
game (with Grundy value equal to the heap size), whilst an individual
block position in game A) is all but trivial.
