At present, it runs primarily on Unix, under either Sicstus 2.1 or
Quintus prolog.  

I have not tried porting it to a Mac yet, though I intend to do this
in the next few months, as I now have a Mac myself!  I think it should
run under quintus-compatible Mac prologs (like LPA Prolog), but I'm
not sure. 

( I am told that Sisctus 2.1 runs on a PC, so PC users
should be able to play Metagame if they get a copy of Sicstus. If
anyone wants to try this let me know.)  

For those who don't know about the Metagame Workbench or the idea of
Metagame, check out the README file on ftp.cl.cam.ac.uk:/users/bdp.
