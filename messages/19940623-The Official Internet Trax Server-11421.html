The "Official" Internet Trax Server is moving!

pbmserv@netcom.com is replacing munch@soda.berkeley.edu as the "Official"
Internet Trax Server.  munch@soda.berkeley.edu will no longer accept new
Trax games/players starting in the near future.  Any existing Trax boards
may be played to completion.  pbmserv@netcom.com is now the Internet Trax
Server.

For more information on the new Trax Server, send E-mail to:
	pbmserv@netcom.com
with a subject line of:
	help

To move a munch board to pbmserv, make sure that both players have signed
up with pbmserv, and then contact Richard Rognlie at rrognlie@netcom.com
to move the board.
