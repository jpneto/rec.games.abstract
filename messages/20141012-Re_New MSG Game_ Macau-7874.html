It seems to me that there is misunderstanding.

As long as any game based on luck give to the 2 players the same probabilities to access to the win victory then it will fair. It will be fair even if the dice are biased. Except if they use different dice of course.

No need for pie rule in games based an randomness. It is obvious.

When a game give an advantage to the first player (like chess game or hex) we could say that there is a random factor at the start of game. So the game is not absolutely determinist. If we could find a way other than pie rule to balance the game like giving the second player the right to choose a set of bad positions to play by the first player it will be fine. The set has to be big enough so no player could be significantly disadvantaged.

So the first player will have to choose between k bad positions defined by the second player. After that the game could start.

In chess game for example the black player could impose to the player not to move the knights and the 2 pawns (king and queen) for example. Even if there are some known openings starting with the pawns (knights). Experts have to decide what are the bad starting positions for the white player.

In hex maybe not to start the game by the central positions and so on.

It will balance for sure the game if and only if the choosen positions are really bad.


Dice, or some other random element are necessary for perfect balance.  Necessary but not sufficient.  As one one countless examples, Backgammon is biased in favor of Player 1.
