On Nov 3, 12:10 pm, Rich Hutnik <getr...@1upandup.com> wrote:


The problem is that
the random setup of neutral elements may create a strong advantage for
the first move player or for the second move player.  In a game like
Go, where you have a measurable advantage for the first move player at
various skill levels you can apply different handicaps at these
various skill levels to level the playing field.  You can't do that in
a game with a random initial setup of neutral elements.  So with
random initial setups of neutral elements, yes, you can have a perfect
information game but the equitableness is dubious at best.


The only way to do this is that it is either symmetric or players play
both sides.  Even then, the fact that a player goes after another
causes an unbalanced position.  So, no matter what, you face a
potential for for an unknown disadvantage.


No.  You're missing the point.  No game is perfectly balanced.  But
something can be known about the move order advantage.  For example
Othello is known to have a second move advantage.  Go is known to have
a first move advantage.  The strength of the first move advantage, as
a function of skill level, is also known in Go.  If you have a game
with a random initial setup, you don't know if there's going to be a
first move advantage or a second move advantage for any particular
random setup and you definitely don't know the strength of the move
order advantage.

For example, say you're moving first in Game X with a random initial
setup.  You may get really lucky and get an initial setup that heavily
favors the first move player.  Or you may have terrible luck and get
an initial setup that heavily favors the second move player.



When you have a wider range of openings, the end result here is that
it is a deeper game.


No.  Wrong.  Depth is a characteristic of gameplay.  Chess is
considered to be a deep game.  Dots and Boxes is considered not to be
a deep game.  Wider is not deeper.

Now, granted, there is some degree of overlap.  In Backgammon, opening
setups with much fewer pieces as compared with opening setups with
much more pieces can result in gameplay of varying depth.  But if you
have a random setup of a fixed number of pieces in Chess, sometimes
that'll result in a deeper game and sometimes it'll result in a less
deep game.  Randomness in the opening setup doesn't automatically
produce a deeper game.


If the game is done electronically, it isn't memorized by people.  The
people end up doing it.  And what the heck is this "judge the game on
its own merits"?  A game can be acceptable and unbalanced, if people
play both sides of a game.  This is a natural way to handle unbalanced
games.



I'm not suggesting including or excluding games based on how balanced
they are.  I'm saying I don't like the idea of using a random initial
setup for the reasons I've outlined.  And I don't think that forcing
players to play both sides of the same randomly set up game is a
solution that magically makes random setups desirable.  It's an
idiotic notion.  There are three possible outcomes to such a scheme:
2-0, 1-1, and 0-2.  A full third of the possible score permutations
result in a tie.  That sucks.  The more unbalanced a game is, whether
as a result of an unfortunate random setup or just because it's a
normally unbalanced game, the more likely the outcome will be 1-1.
