I am not a master of combinatorical game theory, so perhaps someone
could enlighten me. How is the third pile equivalent to 3? I cannot
grab all three stones in one turn. I would expect something similar
to:


The Grundy number of a position is the smallest number that you cannot
move from a position. So if there are moves to 0, 1 and 2, but not 3
then the value of the position is 3. If there are moves to 1 and 2, but
not 0 then the value of the position is 0.
