This seems like maybe an over-strong criterion by which to judge this
effort, if only because I can't imagine any circumstance under which
Go players would ever deem any game better than Go, even if you
created a game that was in some sense actually superior.  Experienced,
dedicated players are both emotionally wedded to their game and can
"see" into it more deeply than they can other games, so they'll always
think it best. The effect is really obvious among chess players since
a few profoundly good variants actually exist, but most chess players
would never admit that they rival FIDE chess for quality.
