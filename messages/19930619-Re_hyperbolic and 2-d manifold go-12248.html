I think the advantages of Penrose tiles for Go can be gotten more
simply by using a single set of 60 degree/120 degree rhombuses
(diamonds).  Such tiles could be used to create a regular grid,
but they could also be used to create an irregular grid with 3 to
six tiles meeting at the vertices.  It might be possible to play
on the tiles or the vertices.  Either way, it would be possible
for the players to create either a traditional board or a board
which deviated from tradition by a small or large degree.  When
playing this game the first few times a set of 81 tiles might be
appropriate.
