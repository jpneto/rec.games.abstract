Beating Mark to it, if the move is a square step, it must be towards
either the row or the column the opponent's queen occupies.
If it is a diagonal step (as in figure 6) it must be towards either
the one diagonal the opponent's queen occupies, or towards the other.
In the 'diagonal case' a move may end up the same distance if (and
only if) the singleton crosses one of these two diagonals.
