That's interesting, it looks as though there is an entire class of
othello-like games, which generalizes othello in the same way that
my class symmetric chess-like games generalizes chess and checkers. 
Does anyone want to try to define this class, and possibly produce a
generator for it?  One interesting consequence would be the need to
produce a more general evaluation function:  How might we generalize
features like STABILITY to apply to different sets of rules?
