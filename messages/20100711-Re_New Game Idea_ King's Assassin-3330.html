I VERY STRONGLY RECOMMEND, for this and any other Quoridor
variants - that the original idea that there be a limited number
of walls to be available for use be *completely abandoned*.

I strongly suspect this rule was introduced merely to save
the game makers some money.  It is highly artificical,
and the number of walls is completely arbitrary.  Having walls
in unlimited supply allows greater opportunity for player skill.
We have found in our played variants that it is far better.
