But if you're interested in GOOD games, how about The Sigma File,
invented by Dr. Eric Solomon? I've never seen the game, but I read about
it in a book of classic abstract games. It's basically a 4-person
boardgame with an international espionage setting. The object is to
use the agents to maneuver the Sigma File document into your home city.
One interesting twist is, you're rarely sure which agents are under your
control, and which will turn on you. Bluffing is a major element.

Yes - I've played sigma file quite a few times; a very tight, well-balanced,

"Adel Verplichtet" - the game of bidding/stealing collectors' items - by

less of a board game & more of a card game.
