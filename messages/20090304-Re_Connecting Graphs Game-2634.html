I think his point was that he composed the diagram using a
proportional-width font, so it looks messed up when viewed
in a fixed-width font.  The correct solution is for HIM to
use a FIXED font when composing the diagram.  There is no
way to use a proportional font and have all your readers see
the intended diagram, because not all proportional fonts
use the same proportions.  Even all fixed fonts aren't the
same, of course, since they may have different aspect ratios,
but at least the topology will be visible in any fixed font.
