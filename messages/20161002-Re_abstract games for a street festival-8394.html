If you like Slither, I suggest trying some of my own square-board connection games. As a designer, I'm kind of obsessed with them. I'd particularly recommend Cation, my current favorite. I think it will prove to be as interesting as Slither if given the chance. Both games feature an unusually high game length versus board size ratio, but Cation achieves that effect without double moves and with only a limited amount of piece movement (pseudo-movement, really).
