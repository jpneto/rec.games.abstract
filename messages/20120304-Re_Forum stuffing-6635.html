Graham, this balancing rule of yours doesn't work, no matter which number (odd or even, greater than 1) of moves are made before the choice of colors. The point is that, if the turn alternation starts before that, the choosing player will always try to make the position as unbalanced as possible in order to maximize his advantage, rather than the opposite.


I think the komi / pass idea is a good one, but I suspect it will be quite hard to balance without a considerable number of games to draw upon.


That's not a problem. You can use the same protocol as in Loose: the first player sets the komi, then the second player chooses sides.

As Bill Spight explained in L19, half-integer komi is also possible by giving Black a "conditional pass", which he can only use if he passes first. This refinement effectively doubles the precision of the pass komi.
