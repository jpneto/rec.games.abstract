This is a relatively easy to learn strategy game that can be played by printing off a hexgrid and using coins or similarly sized markers. It seems to be somewhat unique, but Dots and Boxes is probably it's closest relative.

The rules are explained here: https://www.scribd.com/doc/258354969/Harvest

It's a pretty simple little tactical game. Sometimes the middle-games are a little disappointing, but other times you seem to get some relatively tricky puzzles throughout the whole match.

I mainly made it because a game with only one unit type seemed like a good challenge.
