For some years we had a game that could accommodate
large groups surrounding a theme that we ran at some
conventions.

However it was not abstract in that a theme had to
be applied to the game.  Themes ranged from Fantasy
to Science Fiction.

We also learned that it worked regardless of game
mechanics.  Strangely enough we ended up calling
them "Parallel(tm)" events.

The original group is currently dispersed now and
the leader now works for GAMA.org.  We're hoping
to make a comeback.
