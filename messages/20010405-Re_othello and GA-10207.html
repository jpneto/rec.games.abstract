A simple fitness test is to play games to completion and see which
programs wins most often. To avoid repetition, a number of random
start positions should be tested rather than complete games.
Otherwise, you risk that your population converge to a program that
plays a single line of play perfectly but do miserably on all others.
You should use positions from all stages of the game, from initial
game to endgame.

Additional parameters in the fitness function can be time and memory
usage (smaller == better).
