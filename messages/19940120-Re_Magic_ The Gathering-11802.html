I think its a great game myself.

There is a lot of discussion on this game on:
  rec.games.board
(in fact, it is dominating the group to the extent that there is a lot of 
discussion on forming a group devoted to "MtG")

You can also subscribe to a digest of questions about MtG answered by company 
representatives (Wizards of the Coast) by sending a message to:
  listserv@wizards.com
with the message
  subscribe gg-l <your@net.name>

You can also ftp information on it from marvin.macc.wisc.edu

Note:  the game is an interesting set of rules played with hundreds of 
different unique cards, many of which have the effect of changing or bending 
the rules.  This results in a number of possibly bizarre juxtapositions of rule 
changes / card plays.  The net is mainly devoted to these strange questions.  
The game actually plays much more smoothly than the proliferation of net 
questions would lead you to suspect.
