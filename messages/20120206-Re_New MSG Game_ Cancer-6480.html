Well, that's precisely why I don't like Dots'n'Boxes and its variants.
They're interesting from a mathematical point of view, and I can live
with that because they're as simple as a game can be, but making a
game with groups, liberties, eyes, territories, surrounding capture
and three-color stones behave like a merely parity game seems like a
waste to me. It's like using the Taj-Mahal as a doghouse.


I suspect capricious Cancer outcomes are more likely on a toy, 5x5x5
board.


I don't think so. As in Dots'n'Boxes, toy boards will only help make
the capricious outcomes more or less manageable, but Cancer should aim
higher than that.


How much serious strategy were you hoping to derive from toy
Cancer?  How strategic would toy Go be?


On toy boards, Go isn't deep, but it _is_ strategic.


If the Nimliness of Cancer is really a problem, maybe we *should*
consider non-cancerous eyes.  Completely different game though.


Yeah, I think you should consider that. The game would be far closer
to Go, sure, but at the same time cancer stones and simultaneous
captures would change many life and death situations, and figuring out
how non-cancerous Cancer differs from Go looks like much greater fun
than figuring out how cancerous Cancer differs from Dots'n'Boxes.
There are still some parity situations in the former, but they can
only occur if the game is completely even, which is not the case with
the latter.
