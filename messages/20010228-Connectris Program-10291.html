Hello everybody.

This is a call for beta-testers. Northumbria, the society for the
preservation of board games (http://northumbria.tripod.com) is now
conducting a beta-test for Connectrix Version II. The game combines both
connect f*ur game and a version played on the Richard PBeM server called
Connectris.

 If you like playing abstract games then you might enjoy this one. If you
don't know the games visit Richard web site for rules
(http://www.gamerz.net) or visit the download section of the Northumbria
site and download Connectrix version 1 which is much weaker then the one we
are developing in.

 If you are interested in trying it out and give your comments please email
me on bgalon@amanet.co.il


 P.S. This is not a commercial email. All of the software the society writes
is freeware and we all program for our own love of abstract games!
