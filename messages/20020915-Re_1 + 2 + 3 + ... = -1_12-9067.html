...and also works by the same "divergent telescoping cancellation" as before.

Hmmm...  I wonder if this should be called "Microscoping Cancellation" - it
makes things bigger and bigger as you go along...

Anyway:-   S =  1 + 4 + 16 + 64 +...

Thus  1 + 3S =  1 + 4 + 16 + 64 +...
              + 1 + 4 + 16 + 64 +...
              + 1 + 4 + 16 + 64 +...
              + 1
             =  0 + 0 +  0 +  0 +...   with exact carrying everywhere, no chads.

So  S = - 1/3 ,  as requid.

This is OC the 4-adic calculation writ large, as implied by David.
