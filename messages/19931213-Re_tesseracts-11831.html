I don't usually read this group, but I do know about numbering hypercubes.
Of course you can arbitrarily number the cube, but the normal method, which
makes it easier to work with, it to start with 0, and make each direction
represented by a bit.  Thus connected to 0 would be (in binary)
  0001 0100
    | /
  0000-0010
     \
      1000 (fourth dimension)

by inverting each of the four bits, you get the four neighbors, if you 
want the neighbor in the up direction, invert the bit in the up direction.
0 connects to 1,2,4,8

1 to 0,3,5,9
2 to 0,3,6,10
3 to 1,2,7,11
4 to 5,6,0,12
5 to 4,7,1,13
6 to 7,4,2,14
7 to 6,5,3,15
etc.

If your making these room, then each bit could be wall with a door
and each edge a door.
