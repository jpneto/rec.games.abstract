In summary, I can enumerate the following:  (did I miss any?)
OBJECT checkmate, elimination, racing, territory, configuration, 
connection
Each needs to be further refined, e.g. checkmate/chess vs 
checkmate/capture,
configuration/line/5, configuration/square/any-size, hybrids thereof.

Vocabulary for other attributes:
STRATEGY mobility, position/center, fence ...
TACTICS development, position/boundary ...
CAPTURE displacement(/by-rank), sandwich, leapfrog, surround, strand ...
MOVEMENT adjacent/various, phalanx ...
PLACEMENT array, vacant, adjacent/orthogonal ...
FORMAT square/19, hex/rhombus/11 ...

Obkect: Expulsion
