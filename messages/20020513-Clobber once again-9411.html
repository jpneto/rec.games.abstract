Last week I posted a link of Java-clobber written by one of my game theory 
students to this newsgroup. Some of the people submitted bug reports about 
the applet not being able to run under Microsoft's JVM. I did some 
investigation on the matter and found out that it is not a bug, but rather 
a feature. The problem lies in incompatibility of Microsoft's Java with 
everybody else's and the best fix to the problem would be getting Sun's 
Java plugin to Your IE or (as I do) use Linux with Netscape 6.2. And please 
redirect all Your complaints about Microsoft's JVM to Microsoft.

Still, the author of the program did some adjustments (including 
improvements in English) and the new version can be found at a new address

http://www.cs.ut.ee/~jan/game/ClobberApplet.html
