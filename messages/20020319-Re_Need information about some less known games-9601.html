Joao Neto <jpn@di.fc.ul.pt> wrote:

Does anybody has some information about these games?



Alcatraz by Bernhard Weber


I'm pretty sure this isn't a pure abstract, as there is die-rolling
involved (it's been a while since I've played, so I'd have to check).
It was OK but not that exciting.



Attraction       Publ:Dujardin
Barrier          Publ:Aladdin
Carapace (~1970) Publ:Plan B


This is an unequal forces game in which it's predetermined which side
will win.  The object of the "loser" is to do as well as possible, and
you play in 2-game sets and add the scores.



Crosse           Publ:DMR


This is something like misere checkers - you're trying to get rid
of all your pieces (it's not exactly like it, but I don't remember
the details).



Courier          Publ:Aladdin
Ergo             Publ:Invicta
Kage             Publ:DMR


This game involves placing barriers between spaces to try to trap
your opponent's piece.



King's Court     Publ:Golden


This is a "supercheckers" game with an expanded board (and some
additional rules).

I have copies of Carapace, Crosse, Kage, and King's Court available
for sale, BTW (and possibly Alcatraz as well).
