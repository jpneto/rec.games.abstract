There is a new entry at Richard's PBeM Server: "Accasta"

http://www.gamerz.net/~pbmserv/accasta.html

It is a stacking game on a hexagonal board where you can split up and
combine pieces in many ways. Simple rule set and a huge number of
possible moves result in short and exciting games. Give it a try! If
you like Lasca, Octi, or Abalone you will enjoy the new challenges of
Accasta.

Rules, tutorials and other information at: http://www.accasta.com
(Auf Deutsch: http://www.accasta.de).

You can also check PBeM-Server positions rendered in nice SVG graphics
at http://www.accasta.net.
