Game design IS NOT a science. There is nothing "cumulative" in game design. So we can not stand upon the shoulders of giants in designing games. In science and technology where the knowledge is cumulative yes we have no choice then starting from what was done. There is no shoulders of giants in game design. First because there is no giants in game design. All those called giants did one thing : they summarized what was done! That`s it. Few invented what we call wrongfuly new games. None of those games was really new. 
Each game is specific.
Each mechanics vary depending on which game you applied to.
What is a huge mistake in game X could be improvement in game Y.
Game design is more art than science.
Game design could use science, technology, mathematics, graph theory BUT it has nothing to do with science, technology, mathematics or graph theory.
