Like Martin, I believe an odd full board is a first-player win, but I
will need to think a bit more to find a proof.


The first-player just takes the middle piece and the copy the
second-players moves like in the even full board version.


That doesn't work.  Example:

 Start   1st pl.  2nd pl.   1st pl.  2nd pl. (wins)

  XXX     XXX      XX         X
  XXX     X X      X 
  XXX     XXX      XX         X

Note that the 1st player copied the 2nd player's move, as you
suggested.

I agree that the first player should start by taking the middle stone
(or any single stone, actually, as they are all equivalent).  But the
only correct response to the 2nd player's move is

 Start   1st pl.  2nd pl.   1st pl.

  XXX     XXX      XX        XX
  XXX     X X      X 
  XXX     XXX      XX        XX

as this effectively reduces the board to a 2x2 full board.  Note that
this is in no means a copy of the 2nd player's move.
