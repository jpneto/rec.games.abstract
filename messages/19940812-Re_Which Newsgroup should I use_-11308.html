|> games:
|> /rec/games/abstract

Generic Abstract Games.  Cribbage is probably best discussed here. 

|> /rec/games/board

Generic Board/Paper/Card games not covered by specific other groups.
War games (like Axis and Allies), Railroad games (like 1830 or Eurorails), 
Election games (like Die Macher) are all discussed here.  Generally if a
game generates too much traffic then it is moved to its own group.  
Magic the Gathering overran the group for a while before /rec/games/deckmaster
was generated for it.

Cribbage is also welcome here (IMHO).

|> /rec/games/board/ce

Discussions about the game Cosmis Encounters.  (This is a particular game that
generated too much traffic for rec.games.board.)

|> /rec/games/board/marketplace

Board games for sale.

Also

rec.games.misc

Computer games (I'm not sure why, but that is what is discussed there.)
