"Part of the problem is that you have only five judges - normally I'd
recommend the PR-STV voting system (ask Dec about it, it's the Irish
voting system). If you are only concerned with the top 3 games, the
quota would be too difficult to reach!

My recommendation would be to have a run-off situation. Have people
rank the games. Count how many times each game makes the top 3. The
games that are in the top 3 most frequently are then the "top 3"
games. Then give the list to the voters again and have them rank only
these games. I'd then give 5 points for every top rank, 3 points for
every 2nd rank and 1 point for every 3rd rank. Add up the ranks given
to each game and award the 1,2,3 positions on the basis of how many
points each gets."
