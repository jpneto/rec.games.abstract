Hi everyone,

I am interested in playing E-mail games in the following games:

- English checkers
- Russian checkers
- Brazilian checkers
- International checkers

(I assume English checkers is known to you all. Russian checkers has some
other rules. For instance you can capture backwards, there is a 'long' king,
and as soon as a piece reaches the promotion-line, it becomes a king (even af-
ter a capture!) and can continue with the capture as a king.
Example of the last rule :

+--+--+--+--+--+--+--+--+
|  |  |  |  |  |  |  |  |
+--+--+--+--+--+--+--+--+
|  |  |  |  |  |  | b|  |
+--+--+--+--+--+--+--+--+
|  |  |  | b|  |  |  | w|
+--+--+--+--+--+--+--+--+
|  |  |  |  |  |  |  |  |

White can capture h6*c5.

Brazilian checkers : majority capture is obliged, capture backwards is pos-
sible, and there is a 'long' king.

International checkers : Same as brazilian checkers, only played on a 10*10
board.
