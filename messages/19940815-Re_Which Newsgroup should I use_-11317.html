Leaving out the definition of abstract game, alas.  Roughly:

	An abstract game has no hidden information and no random
	events.

So chess, checkers, nim, tic-tac-toe are abstract games.  Backgammon is
not (random events).  Twenty questions is not (hidden information).


Generic Abstract Games.  Cribbage is probably best discussed here. 


Cribbage is not an abstract game.  But hey, it's o.k. with me if you
talk about cribbage in rec.games.abstract. :-)  Despite the intent when
this group was created, true abstract games have typically been a small
part of the discussion here, and nobody seems too upset.
