If you started with (2,11) then it would matter. My interpretation is that
player 1 loses because there are no legal moves from any position that
contains only primes. He can't add 2+11 and pass the lone 13 to player 2.

But if you start with (3,10) then player 1 is allowed to do 3+10=13 and win,
because player 2 can't do anything with the 13.
