The ContraPloy site has been upgraded:
http://www.contraploy.com

The game is now completely browser based.  No download is necessary. 
Other features that have been added since my last post: player list,
automatic player ranking, game log, message board, and player mail.

Disclaimer: this game violates the technical definition of abstract:
though it harbors no arbitrary randomness, a player's piece values are
hidden from the opponent.  In exchange for committing this one sin,
the game offers an excellent balance of depth, clarity, drama and
decisiveness.  The kind of experience only an abstract gamer can
really appreciate.
