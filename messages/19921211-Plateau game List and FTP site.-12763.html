There is a mailing list for the game Plateau.  For information
and to subscribe to the list contact:

                 plateau@plateau.com

                         or

              ...uunet!plateau!plateau


An electronic version of the game (called "Honor Plateau") that
facilitates email play is available via anonymous FTP at:

                ftp.cd.chalmers.se

                 Name: anonymous
             password: <your address here>

                 Next: $ uncompress plateau.tar.Z
                 Then: tar xvf plateau.tar
