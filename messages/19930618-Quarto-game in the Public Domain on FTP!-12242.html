Hello all,

A while ago, I announced that I would release a program that would play
Quarto!. Now, it is finished, and it has been put on the rec.games.abstract
FTP-site, pprg.eece.unm.edu, in directory pub/rec.games.abstract/games/quarto.

The program is for Windows 3.x, and it requires the VBRUN200.DLL to be in
your /windows/system directory. VBRUN200.DLL is freeware, and I bet it can be
ftp'ed from a lot of ftp-sites.

I think the program is much self-explaining, and I made a help-screen as well.

If you have remarks, or if you could supply me with a better algorithm, please
mail me, at heinr@sci.kun.nl.
I look forward to your comment.

Greetings,
