  Y is a very simple game to explain to anyone.  I've been able to bring 
it to college, and find people willing to play it.  That's a good trait 
for an abstract game to have.  If you can never find an opponent, it 
doesn't matter how good the game is.

  As a strategy game, I rate it rather highly.  I especially like the 
varient where you can't play adjacent to the last move played.

  I've come up with a notation system that seems to work rather well.  
The game in the booklet (Figure 32) would start out 

A5  C9  H4  D5  D7  G6  E10  G8  F10  H8  E6  E2  D3  E4  G5  H3  G3  F3  D4

  Still, I think I like Go and Amazons better.  I've had luck finding 
opponents for Amazons, too... :)
