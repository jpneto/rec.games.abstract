Hello Othello Friends,

we have installed an Internet Othello Server (IOS) here in Paderborn (Germany)
reachable via "telnet faust.uni-paderborn.de 5000" or by running "olios",
the OpenLook interface to the server. You can get this via 

  ftp ftp.uni-paderborn.de    (user: anonymous, passwd: guest)
  cd unix/othello
  ...

In this directory you may find Othello pd-software, tournament results etc.
