Whenever I read about a new game I'd like to play,
I create the game board in xfig format.

I have the following 5 boards:
   backgammon (not an `abstract' game?)
   chinese checkers
   focus (Sid Sackson's)
   19x19 grid for go, pente, etc
   peg solitaire

Do people feel that these "soft" versions of the game boards would be useful?
Should they be part of the archive?

If xfig is not an acceptable format, they can be converted to postscript.
However, they couldn't be edited (very easily).

I guess the only problem I've had with these "soft" game boards
is that they get printed out on 8.5x11" paper which is too small
for chinese checkers, go, etc.  So I take it to a photo copier
to copy it onto larger paper.  The 8.5x11 paper is fine for
backgammon and peg solitaire, tho'.
