Added new logic game: Breaker of blocks

Program Title: Breaker of blocks
Description: 

   This is a logic arcade game made on the combine puzzle and action
style game. The idea of the game is to move the blocks around the
screen until 2 or more of the same block touch. Once this is done they
disappear. Once all the squares have gone and the round cleared, the
game progresses to the next level. Each level has a time limit. Heres
a round for you to look at!
   Easy to learn, but very tough to master! The game helps to spend
your time well and useful. It is interesting and addictive for both
children and adult; for office and home computer.
