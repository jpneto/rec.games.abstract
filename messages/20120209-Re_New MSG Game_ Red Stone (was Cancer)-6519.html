Would there ever be a situation where you'd want to sacrifice a group
like that?


Yes, there would. For example:

.. o . o x x
o o o o x x
o o o o x x
o o o . x .
# # # o x x
x x x . x .

Black (x) to play.

If 1. xd1, od3 is a White win, whereas if 1. #d1, all the lines are
Black wins, for example:

1. ... ob1;
2. #d3, oa1;
3. #c1, oa1;
4. #b1 and wins.

Not the ideal example, since Black can also win with 1. xd3, but you
get the idea.
