For games played between experts, I recommend viewing the annual world
chamionship games:
http://arimaa.com/arimaa/wc/

please judge for yourself whether these appear to be drawish.

If time controls are not to your liking, I recommend looking at the
postal games.  A good place to start is the 2009 postal mixer in which
a player could take hours or days for each more:
http://arimaa.com/arimaa/tourn/postal/2009/showGames.cgi

The 2010 postal mixer is underway currently:
http://arimaa.com/arimaa/tourn/postal/2010/showGames.cgi

For a more interesting perspective, I recommend viewing some of the
commented games.  This is a good example:
http://arimaa.com/arimaa/video/go/chessandgo2009WC.htm
There are many more commented games that can be found at the Arimaa
web site.

Currently, the draw rate for Arimaa is 0%.  It used to be possible to
reach a draw, but there was a rule change a few years back that
resolved that, and now there just aren't any draws.  You can check the
game history on the site to check this fact.
