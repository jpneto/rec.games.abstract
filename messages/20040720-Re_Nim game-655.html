I think that he means Grundy number. If a position has the number n
then it is equivalent to a simple nim pile of n matches.
http://encyclopedia.thefreedictionary.com/Sprague-Grundy%20theory

I am not a master of combinatorical game theory, so perhaps someone
could enlighten me. How is the third pile equivalent to 3? I cannot
grab all three stones in one turn. I would expect something similar
to:

(.) -> 0

(X) -> 1

(X X) -> 2

(X X X) -> 3

(X X)  
(X .) ->  <1,2|1,2>

Here the number <1,2|1,2> is surreal. It means that if left player
moves first he can leave a simple nim pile of size 1 or a simple nim
pile of size 2. The same goes for the right player.
http://en.wikipedia.org/wiki/Surreal_number
